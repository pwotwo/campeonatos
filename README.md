# 🏆 Sistema de Gestão de Campeonatos Desportivos

Sistema completo para gerir campeonatos desportivos — inscrições, jogos, classificações e estatísticas.

## 🛠️ Stack Tecnológica

- **Frontend:** React + TypeScript + Vite + Tailwind CSS
- **Backend:** Node.js + Express + TypeScript
- **Base de Dados:** PostgreSQL
- **ORM:** Prisma
- **Autenticação:** JWT

## 📋 Funcionalidades

- Gestão de campeonatos (múltiplos formatos)
- Inscrição de equipas e jogadores
- Registo de jogos e eventos em tempo real
- Classificações automáticas
- Sistema de sanções e disciplina
- Estatísticas por jogador e equipa

## 🚀 Como Instalar

### Pré-requisitos
- Node.js 18+
- Docker Desktop

### Passos
```bash
# Clonar o repositório
git clone https://github.com/pwotwo/campeonatos.git
cd campeonatos

# Arrancar a base de dados
docker-compose up -d

# Instalar o backend
cd backend
npm install

# Instalar o frontend
cd ../frontend
npm install
```
